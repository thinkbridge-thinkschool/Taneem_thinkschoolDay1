Day 7 — Window functions
=========================

## SQL Query

SELECT
    Author,
    Text,
    ROW_NUMBER() OVER (PARTITION BY Author ORDER BY Id)           AS QuoteNumber,
    SUM(1)       OVER (PARTITION BY Author ORDER BY Id
                       ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW) AS RunningTotal,
    Id - LAG(Id) OVER (PARTITION BY Author ORDER BY Id)           AS IdGapSinceLastQuote
FROM Quotes
WHERE IsDeleted = 0
ORDER BY Author, Id;

## Note on "gap in days"

The Quotes table has no CreatedAt column, so there is no real date to diff.
Id gap (current Id minus previous Id via LAG) is used as a proxy — it shows
how many rows apart two consecutive quotes from the same author are in the table.
A gap of 1 means they were inserted back-to-back; a larger gap means other
authors' quotes were inserted in between.

## What each window function does

- ROW_NUMBER() OVER (PARTITION BY Author ORDER BY Id)
  Numbers each quote per author starting from 1. Resets for each author.

- SUM(1) OVER (PARTITION BY Author ORDER BY Id ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW)
  Running count of quotes per author up to the current row.

- Id - LAG(Id) OVER (PARTITION BY Author ORDER BY Id)
  Gap proxy: difference between the current row's Id and the previous row's Id
  for the same author. First quote per author is NULL (no previous row).
