Day 5 — Add Polly resilience to HTTP calls
===========================================

1) HttpClient + resilience handler config (ServiceExtensions.cs)

services.AddHttpClient("external-quotes", c =>
{
    c.BaseAddress = new Uri(configuration["ExternalQuotes:BaseUrl"] ?? "https://zenquotes.io");
    c.Timeout     = Timeout.InfiniteTimeSpan;
})
.AddResilienceHandler("default", (builder, context) =>
{
    builder.AddRetry(new HttpRetryStrategyOptions
    {
        MaxRetryAttempts = 3,
        BackoffType      = DelayBackoffType.Exponential,
        UseJitter        = true,
        OnRetry = args =>
        {
            logger.LogWarning("Retry {Attempt} after {Delay}ms — {Outcome}",
                args.AttemptNumber + 1,
                args.RetryDelay.TotalMilliseconds,
                args.Outcome.Exception?.Message);
            return ValueTask.CompletedTask;
        }
    });

    builder.AddCircuitBreaker(new HttpCircuitBreakerStrategyOptions
    {
        SamplingDuration  = TimeSpan.FromSeconds(30),
        FailureRatio      = 0.5,
        MinimumThroughput = 3,
        BreakDuration     = TimeSpan.FromSeconds(15),
        OnOpened = args =>
        {
            logger.LogError("Circuit breaker opened for {Duration}s", args.BreakDuration.TotalSeconds);
            return ValueTask.CompletedTask;
        }
    });

    builder.AddTimeout(TimeSpan.FromSeconds(10));
});

2) Retry logs (forced failure with bad URL)

[WRN] Retry 1 after 1906ms — No such host is known. (this-host-does-not-exist-xyz.io:443)
[WRN] Retry 2 after 1436ms — No such host is known. (this-host-does-not-exist-xyz.io:443)
[ERR] Circuit breaker opened for 15s
[WRN] Retry 3 after 3000ms — No such host is known. (this-host-does-not-exist-xyz.io:443)
[ERR] The circuit is now open and is not allowing calls

3) What happened

1. Request failed on first attempt (DNS resolution failed)
2. Polly retried 3 times with exponential backoff + jitter
3. After 3 failures, circuit breaker opened for 15 seconds
4. 4th attempt was blocked immediately by the open circuit
5. BrokenCircuitException returned to caller — no more network calls wasted
